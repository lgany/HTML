HTML

이 페이지는 HTML에 관한 내용입니다.

HTML은 웹 문서를 만들기 위해 사용되는 기본적인 웹 언어의 한 종류로서 모든 웹페이지가 HTML로 되어있다.

인터넷에 기본적인 실행 방법? 인터넷을 정상적으로 실행하려면 기본적으로 컴퓨터 2대가 필요하다
1. 웹브라우저 (고객) 2.웹 서버(기업)
고객이 웹 브라우저를 통해 어떤 주소를 요청 하면 웹 서버가 응답하여html로 작성된 코드를 특정 디렉토리에서 가져와서 실행시킨다.

하지만 웹서버를 직접 운영하는 일은 힘들기 때문에 웹호스팅 업체를 통해 사용되는 경우가 더 많다.
1. 클라이언트 2. 웹 호스팅업체(기업) 3. 방문자(도메인 접속원하는사람)
1번이 만든 웹페이지에 3번이 접속할수 있게 만들고 싶다. 그러면 1번은 2번이 만든 웹페이지에 자신이 만든 웹페이지를 올려서 접속하게 만들수 있다
1번이 한번 올려두면 2번(서버)과 3번(클라이언트.)만 서로 응답과 요청을 하는것임.
