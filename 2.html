*CSS

예전의 HTML은 디자인이 없었지만 CSS의 탄생으로 인해 웹페이지에 디자인을 넣는게 가능해졌음 웹페이지 입장에서 어떤 언어가 CSS가 무엇인지 구분하기 위해서 두가지 방법이 있음. 두 가지 모두 style 태그를 사용한다.

1. Style 효과와 선택자 사용.
style a { color:black; <ㅡㅡㅡㅡㅡ color black 은 효과(선언,declation) 라고 한다. } 여기서 효과만 있어서는 어떤곳에 css 속성을 입힐지 모르니깐 a {}라는 선택자가 같이 사용된다.

2.Style 효과만 사용
(li) (a href="2.html" target="_blank" title="2.html" style=color:red) CSS (/a) (/li) 여기서는 선택자가 없어도 어디에 적용할지 알수 있기 때문에 style 효과만 사용한다.
